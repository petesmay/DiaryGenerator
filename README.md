Diary Generator
---------------

Creates a text file containing weekday entries for the specified year, e.g. for 2015, the document starts:

```
15-01-01 Thursday:

15-01-02 Friday:

--------------------

15-01-05 Monday:

15-01-06 Tuesday:

15-01-07 Wednesday:

15-01-08 Thursday:

15-01-09 Friday:

--------------------
```

## Usage

```bash
python DiaryMaker <year>
```
